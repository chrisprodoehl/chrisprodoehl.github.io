Web Csound + Web GL

Real-time audio responsive graphics with Web Csound and Web GL. When playing Csound (not from the file), you can move the slider to affect grain density (higher values = denser grains). You can tap and drag along the x-axis to rotate the camera.

**Real-time Csound is basically unlistenable on mobile, so try playing the file instead. The effect is less fine-grained (rms on the final mix, rather than individual instruments), but better than hearing noise!
